## Installing

Using npm:

[![npm](https://img.shields.io/badge/npm-connect--to-brightgreen)](https://www.npmjs.com/package/chart-it/)

```bash
$ npm install chart-it
```

Then import it:

```html
<script src="./node_modules/chart-it/dist/chart-it.js" defer></script>
```
