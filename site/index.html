Chart Ease

Customizable and high-performance web component to draw static charts.

Chart-it is a lightweight and highly customizable web component designed for creating small and precise charts. With its ability to normalize data from any range to a specified size, Chart-it makes it easier than ever to represent data in an elegant and effective way. Whether you're creating a simple line graph or a complex scatterplot, Chart-It makes it easy to build and display your data in a clear and meaningful way.

CHART PATH

custom PATH TYPE

Define a path for your chart and Customize it

With <chart-ease>, you can customize your chart's path by defining a <path> element inside the <data-set> element and setting the is attribute to "path-type". This will draw the `path` element as your chart's path. 

<path is="path-type" stroke="aqua" fill="none"></path>

Here's a basic example that creates a chart with a path connecting the data points: 

<chart-ease width="200" height="200">
  <data-set>
    <path is="path-type" stroke="aqua" fill="none"></path>
  </data-set>
</chart-ease>
<script src="../js/chart-ease.js"></script>
<script>
  document.querySelector("chart-ease").data = [{ x: 1, y: 6 },{ x: 2, y: 4 },{ x: 3, y: 0 },{ x: 4, y: 3 },{ x: 5, y: 5 },{ x: 6, y: 2 },{ x: 7, y: 7 },{ x: 8, y: 3 }];
</script>

Chart Points

Custom Point Types

Define Custom Point Types for Your Chart Points

You can define any type of SVG element or combination of them as point types. Basic elements are adjusted by default, so you only need to put them inside the same data-set element. Chart-ease will draw them at the corresponding position of each point.

Circle

Use the <circle> element to create circular points. 

                <chart-ease>
  <data-set>
    <path is="path-type" stroke="aqua" fill="none"></path>
    <circle is="point-type" r="5" fill="aliceblue"></circle>
  </data-set>
</chart-ease>
Rectangle

Use the <rect> element to create rectangular points. 

                <chart-ease>
  <data-set>
    <path is="path-type" stroke="aqua" fill="none"></path>
    <rect rx="0" ry="10" width="10" height="10" stroke="aqua" fill="aliceblue" />
  </data-set>
</chart-ease>
Ellipse

Use the <ellipse> element to create elliptical points. 

                <chart-ease>
  <data-set>
    <path is="path-type" stroke="aqua" fill="none"></path>
    <ellipse rx="5" ry="10" fill="aliceblue" stroke="aqua" />
  </data-set>
</chart-ease>

complex chart points

Adjust point types

Define complex Point Types for Your Chart

To create complex point types, you may need to adjust the position, styles, animations, and more. With the `draw` event, you can access each point and draw it yourself. In the ondraw event handler, you can create different types of charts and customize everything.

draw event

<chart-ease> fires a draw event for each point type and passes an object containing the following items to the event handler:

  • shape: the point element, which can be any SVG element type.
  • originalRow: the corresponding row of the data array.
  • row: the normalized row of the data array that is computed based on defined or default axes.
  • index: the index of the row inside the data array.

See these examples

samples 1: Value labels

Drawing the value upon each points using <text></text> and customize it based on index. 

                <chart-ease id="textType" width="200" height="200">
  <data-set>
    <path
      is="path-type"
      stroke="aqua"
      stroke-width="2.5"
      fill="none"
    ></path>
    <text x="0" y="15" fill="black" font-size="10px"></text>
  </data-set>
</chart-ease>

<script>
  const textType = document.querySelector("#textType");
  textType.ondraw = ({ shape, row, index, originalRow }) => {
    shape.setAttribute("x", row.x - originalRow.y.toString().length * 2.5);
    shape.setAttribute("y", row.y);
    shape.innerHTML = originalRow.y;
    if (index === 0) shape.setAttribute("opacity", 0.1);
    if (index === 1) shape.setAttribute("opacity", 0.5);
  };
  textType.data = [{ x: 1, y: 0.6152156 },{ x: 2, y: 0.6152154 },{ x: 3, y: 0.615215 },{ x: 4, y: 0.6152153 },{ x: 5, y: 0.6152155 },{ x: 6, y: 0.6152152 },{ x: 7, y: 0.6152157 },{ x: 8, y: 0.6152153 },
  ];
</script>
sample 2: Custom path

                <chart-ease id="pathType" width="200" height="200">
  <data-set>
    <path is="path-type" stroke="aqua" stroke-width="2.5" fill="none"></path>
    <path fill="none" stroke="blue" stroke-width="2" />
  </data-set>
</chart-ease>
<script>
  window.onload = function () {
    const pathType = document.querySelector("#pathType");
    pathType.ondraw = ({ shape, row, index }) => {
      const { x, y } = row;
      shape.setAttribute(
        "d",
        `M${x - 4},${y - 4} V${y + 4} H${x + 4} V${y - 4} H${x - 4}`
      );
      if (index === 0) shape.setAttribute("opacity", 0.1);
    };
    pathType.data = [{ x: 1, y: 6 },{ x: 2, y: 4 },{ x: 3, y: 0 },{ x: 4, y: 3 },{ x: 5, y: 5 },{ x: 6, y: 2 },{ x: 7, y: 7 },{ x: 8, y: 3 }];
  };
</script>

SAMPLES

Bar chart

Create a custom bar chart

With <chart-ease>, you can create diverse and custom bar charts by using your data with a custom structure and drawing the chart in your own way. In the following sample, we use the <g> element to group multiple elements for each point and draw the values of "Silver", "Gold", and "Platinum". Then we adjust the bars by utilizing the ondraw event handler. 

<chart-ease id="bar1">
  <data-set>
    <g>
      <rect for="Silver" fill="red"></rect>
      <rect for="Gold" fill="blue"></rect>
      <rect for="Platinum" fill="green"></rect>
    </g>
  </data-set>
</chart-ease>
<script src="../../js/chart-ease.js"></script>
<script>
  const data = [
    { x: 1, Silver: 2, Gold: 5, Platinum: 5 },
    { x: 2, Silver: 5, Gold: 7, Platinum: 6 },
    { x: 3, Silver: 9, Gold: 3, Platinum: 8 },
    { x: 4, Silver: 2, Gold: 7, Platinum: 0 },
    { x: 5, Silver: 9, Gold: 2, Platinum: 2 },
    { x: 6, Silver: 2, Gold: 9, Platinum: 3 },
    { x: 7, Silver: 9, Gold: 1, Platinum: 4 },
    { x: 8, Silver: 1, Gold: 3, Platinum: 2 },
  ];
  const bar1 = document.querySelector("#bar1");
  bar1.ondraw = ({ shape, row }) => {
    const setBar = (bar, x, y) => {
      bar.setAttribute("width", 10);
      bar.setAttribute("height", y);
      bar.setAttribute("x", x);
      bar.setAttribute("y", 200 - y);
    };
    Array.from(shape.children).forEach((bar) => {
      setBar(bar, row.x, row[bar.getAttribute("for")]);
    });
  };
  bar1.axes = [
    {
      cols: ["x"],
      lowerBound: 10,
      upperBound: 190,
      length: 200,
    },
    {
      cols: ["Silver", "Gold", "Platinum"],
      lowerBound: 10,
      upperBound: 190,
      length: 200,
    },
  ];
  bar1.data = data;
</script>
